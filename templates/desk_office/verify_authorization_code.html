Verification Guide
Code Status Meanings:
  • Valid: Code is active and can be used
  • Expired: Code has passed its expiry date
  • Invalid: Code is cancelled, used up, or doesn't exist
Code Format:

AUTH-YYYYMMDD-XXXXXX

Where YYYYMMDD is the generation date and XXXXXX is a unique identifier.

Common Issues:
  • Check for typos in the code
  • Verify the code hasn't expired
  • Ensure the code hasn't been cancelled
  • Check if the remaining amount is sufficient
