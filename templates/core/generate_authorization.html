Help & Guidelines
Authorization Code Purpose

Authorization codes are required for NHIA patients accessing services in non-NHIA units or specialty services.

Code Type Options

Auto-Generate: System creates a unique code automatically (e.g., AUTH-20251005-ABC123)

Manual Input: Enter your own custom code (e.g., NHIA-2025-001, CUSTOM-CODE-123)

Amount Covered

Enter the maximum amount that NHIA will cover for this service. The patient may be responsible for any amount exceeding this.

Validity Period

Choose an appropriate validity period based on the service type and expected completion time.

After Generation

The authorization code will be:

  • Automatically linked to this record
  • Visible to relevant staff
  • Tracked in the system
  • Marked as used when service is completed
  • Accepted at all requesting units/departments
