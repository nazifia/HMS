Instructions

How to generate an authorization code:

  1. Search for the NHIA patient by name or ID
  2. Select the patient from the dropdown
  3. Choose the service type the code is for
  4. Enter the amount to be covered by the authorization
  5. Set an appropriate expiry date
  6. Add any relevant notes
  7. Click "Generate Authorization Code"

Authorization codes are automatically generated with a unique identifier.
