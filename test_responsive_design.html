Responsive Design Test Page

This page tests the responsive design implementation across different screen sizes.

Instructions: Resize your browser window or use developer tools to test different screen sizes. The indicator in the top-right corner shows the current breakpoint.

Desktop View (992px+)

On desktop screens, you should see:

  • Full sidebar navigation
  • 3-column card layout
  • Large patient profile images
  • Full-width tables
  • Hover effects on cards

Tablet View (768px-991px)

On tablet screens, you should see:

  • Collapsed sidebar
  • 2-column card layout
  • Medium-sized elements
  • Responsive tables
  • Touch-friendly buttons

Mobile View (767px and below)

On mobile screens, you should see:

  • Hidden/overlay sidebar
  • Single-column layout
  • Smaller text and images
  • Touch-optimized buttons
  • Stacked navigation
